## 📌 Overview

**RE:ORG** is the official landing page and interactive diagnostic platform for SapientHR's premier organizational restructuring program. The application serves as the digital front door for enterprises looking to transition from rigid, hierarchical operating models to fluid, high-performance structures.

The site highlights the **SHIFT Framework** and includes an interactive **Readiness Assessment (The Rigidity Radar)** that dynamically evaluates an organization's structural friction and operational agility.

## 🚀 Features

- **Clinical & Minimalist Design:** High-contrast, premium aesthetic built with Tailwind CSS v4.
- **Dynamic Rigidity Radar:** A custom-built, interactive 9-point SVG radar chart that uses RAG (Red-Amber-Green) logic to dynamically visualize an organization's operational health based on user input.
- **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop viewports.
- **Fluid Animations:** Scroll-triggered reveals and mounting animations powered by Framer Motion.

## 🛠️ Tech Stack

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4 (configured via `@tailwindcss/postcss`)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Utilities:** clsx, tailwind-merge

## 💻 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/JaySree-S-S/Re-Org.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Re-Org
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

Start the local development server:

```bash
npm run dev
```

The application is configured to run strictly on **port 5177**. Open `http://localhost:5177` in your browser to view it.

## 📜 Available Scripts

- `npm run dev` - Starts the Vite development server.
- `npm run build` - Builds the application for production into the `dist` folder.
- `npm run preview` - Boots up a local server to preview the production build.
- `npm run lint` - Runs ESLint to check for code quality issues.

## 🏢 About the RE:ORG Program

RE:ORG represents a cultural and operational shift that enables organizations to respond faster to change and sustain long-term growth. The program targets **Vertical Fatigue**, **Information Silos**, and **Role Rigidity** using the 5-pillar **SHIFT Framework**:

1. **S**CAN (Audit & Diagnosis)
2. **H**ARMONIZE (Align Strategy & Structure)
3. **I**NNOVATE (Design Modern Operating Model)
4. **F**ACILITATE (Manage Change & Transition)
5. **T**RACK (Monitor & Stabilize)
